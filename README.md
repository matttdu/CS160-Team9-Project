BinSight is an application that allows people to see locations for nearby trash, recycling,
and compost bins, as well as the ability to place their own pins, as well as rate existing pins.
BinSight also has social features that allow people to post and reply to other people's posts,
useful for organizing group clean-ups and similar events.

Installation:
The provided APK is able to be installed on any Android device/emulator (i.e., through Android Studio)

Features:
* Log in to an existing account or create a new account
* See nearby pins, rate them, and place your own pins
* Talk with other users through the community tab
* See various stats such as the number of pins rated in the profile tab
